## Re-exported from Pydantic

`pydynantic` re-exports Pydantic's [`Field`](https://docs.pydantic.dev/latest/api/fields/#pydantic.fields.Field)
for convenience, so you can declare entity attribute metadata without a second import. It
behaves exactly like `pydantic.Field`; see the linked Pydantic docs for every parameter.

::: pydantic.Field
    options:
      show_docstring_description: false
      show_docstring_parameters: false
      show_docstring_returns: false
      show_docstring_raises: false
      show_docstring_examples: false
